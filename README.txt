Opizo Link Shortener and Make money system from Links and URLs

== Description ==
<a href='http://opizo.com'>OPIZO</a> Free URL forwarding service (URL redirection) allowing anyone to shorten any existing URL You can custom your link - set password, number of uses, expire date.
<a href='http://opizo.com'>Opizo</a> is a system for shrink urls and make money from shrinked links.
This plugin provided easy way to use opizo services in wordpress sites and blogs.

= Features: =
* Easily Active or Deactive and return original urls.
* Shrink urls in posts and convert urls to revenue.
* Shrink urls in old and new posts.
* Support Post Types.
* Shrink urls directly.
* Income statics chart in dashboard


= Translations: =
* English

== Installation ==

1. Upload `opizo.zip` to the `/wp-content/plugins/` directory
2. Unzip it.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. A meta box add to new post screen.
*. You must register in <a href='http://opizo.com'>http://opizo.com</a> and get an API Code.

== Frequently Asked Questions ==
= It possible return back original urls any time =
Yes, deactive plugin in Settings page.


= I can convert urls in my old posts =
Yes, For shrinking URLs in old posts, go to Post lists, select desired posts then select "Shrink links by Opizo" from Bulk Actions, then go to "Shrink Old Posts" page and press "Start Shrinking" button.
